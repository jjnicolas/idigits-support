Desafío de cálculo mental

Cómo jugar

Combina seis números usando +, −, ×, y ÷ para alcanzar el número objetivo. Selecciona un número, elige una operación y luego selecciona otro número. Encadena operaciones para alcanzar el objetivo en la menor cantidad de movimientos posible.

Consejos

  • Cada partida tiene 5 etapas con dificultad creciente.
  • Gana hasta 3 estrellas por etapa — 3 estrellas significa que alcanzaste el objetivo exactamente.
  • Usa el botón deshacer para probar diferentes enfoques.
  • Cambia entre el modo Fácil y Difícil en Ajustes → iDigits.
  • Construye una racha diaria para ganar escudos, que te permiten ver las soluciones sin perder puntos.
