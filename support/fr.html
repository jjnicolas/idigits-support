iDigits

Défi de calcul mental

Comment jouer

Combinez six nombres en utilisant +, −, ×, et ÷ pour atteindre le nombre cible. Sélectionnez un nombre, choisissez une opération, puis sélectionnez un autre nombre. Enchaînez les opérations pour atteindre l'objectif en un minimum de coups.

Conseils

  • Chaque partie comporte 5 étapes de difficulté croissante.
  • Gagnez jusqu'à 3 étoiles par étape — 3 étoiles signifie que vous avez atteint la cible exactement.
  • Utilisez le bouton annuler pour essayer différentes approches.
  • Basculez entre le mode Facile et Difficile dans Réglages → iDigits.
  • Construisez une série quotidienne pour gagner des boucliers, qui vous permettent de voir les solutions sans perdre de points.
